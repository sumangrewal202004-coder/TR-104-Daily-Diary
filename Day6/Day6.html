Day 6 Learning Summary

Introduction to Bootstrap

On Day 6, I started learning Bootstrap, a popular front-end framework used to create responsive and mobile-first websites quickly. The session focused on understanding why Bootstrap is used and how it simplifies web development.

Bootstrap Setup

I learned how to include Bootstrap in a project using CDN links. The importance of linking Bootstrap CSS and JavaScript files correctly was explained.

Concepts Learned

  • What Bootstrap is and its advantages
  • Bootstrap CDN and file structure
  • Container and container-fluid
  • Bootstrap grid system (rows and columns)
  • Basic typography and utility classes

This session helped me understand the foundation of Bootstrap and how it can be used to build responsive layouts efficiently.
